About / Legal information

Linux-Netbook.com is about mobile computers that run an operating system based on the Linux kernel. The main focus are small laptops a.k.a. netbooks, subnotebooks, or umpcs as well as Chromebooks.

The website is divided into several main sections. In the netbooks and Chromebooks sections you find technical specifications of laptops, that are sold with a pre-installed Linux based operating system. Moreover you can compare all laptops or Linux netbooks or Chromebooks only.

The video section features video reviews and demonstrations uploaded to video sharing sites by users and manufacturers.

In the Linux section you find the Linux distributions featured on this website, a Linux history timeline and the bi-weekly Bits of Linux round-up.

To stay up to date with new content on Linux-Netbook.com you can subscribe to the RSS newsfeed and follow linuxnetbook on Facebook.

Who runs this Site?

Linux-Netbook.com is created and run by Ramiro Gómez. I first got in touch with Linux when I registered my first e-mail account in the mid 90ies, which was provided by the Technical University of Berlin, where I studied.

I had the choice between a Unix or a Linux account and opted for Linux, because I liked the idea of a free and open operating system. Since then I have been a user and advocate of open source software.

I created my first own open source project in 2002, which was followed by several others and contributions to open source projects founded by other people. Check out my GitHub profile to learn more about some of these projects.
