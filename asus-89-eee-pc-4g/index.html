ASUS 8.9-inch Eee PC 4G

Brand: ASUS
Operating System: Xandros
Display: 8.9" TFT LCD (1024x600)
Dimensions (WxDxH): 8.9" x 6.5" x 0.9"
Weight: 2.2 lbs
Processor: 0.9 GHz Intel Celeron
Memory: 512 MB DDR2 SDRAM
Storage: 4 GB SSD
Battery life: 2 hours
WLAN: 802.11b/g Wireless LAN
LAN: 10/100 Mbit Ethernet
Colors: Black, Pearl White
Input Devices: Keyboard, Touchpad, Microphone
Connectivity: USB, SD Card reader

The 8.9" Eee 4G from ASUS is a highly portable and affordable netbook for everyone who wants to run net-centric applications on the move. It comes with an 8.9 inch display with a maximum screen resolution of 1024 x 600 pixels. Still it only weighs 2.18 lbs (0.99 kg) thanks to it's ultra-compact design.

The 8.9" Eee PC 4G is a store specific model identified as Eee PC 900 on the packaging. It is equipped with an Intel Mobile Processor, 4GB Solid State Drive, 512MB DDR2 Memory, 3 USB 2.0 ports, a memory card reader, a built-in microphone, speakers, audio jack, a 4-cell battery, 802.11b/g Wi-Fi LAN, and a 10/100 Ethernet LAN Network Card.

The Linux operating system features more than 50 preinstalled applications for gaming, entertainment, communication, and productivity, such as OpenOffice.org, the Firefox Web browser, and the award-winning email client Thunderbird.
