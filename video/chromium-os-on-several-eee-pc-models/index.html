Chromium OS on Several Eee PC Models

About a year from now Chrome OS is expected to be ready for consumer use. Whether it will ship on netbooks as the primary operating system, as an additional instant-on OS on dual-boot systems, or even on an ad supported Google netbook remains to be seen.

So far you can build your own Chromium-Based OS or run an existing image built by someone else. If you prefer watching a demo first, check out these videos of Chromium OS builds running on 3 different ASUS Eee PC models, namely ASUS Eee PC 1000HA, ASUS Eee PC Touch T91, and the venerable ASUS Eee PC 701 netbook.
