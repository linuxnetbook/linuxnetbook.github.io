PRIZM WM8650 Powered Netbook Running Arch Linux

Whenever I see people tinkering with Linux like in this video, it's a confirmation that operating systems based on the Linux kernel are the most fascinating and fun to work with. Here we see Arch Linux running on a cheap 7" netbook from China, that came pre-installed with Android 2.2 and costed around €50 (~$60-$70).

The author of the video also wrote a post on his Kernel Hacks blog on how he got Arch Linux running on his netbook. He skipped the kernel build process, so if you want to get your hands dirty check out porting Arch Linux to a WM8650 tablet as well.

A search for WM8650 netbook on eBay brings up several potentially interesting results.
