Installing Kuki Linux on the Acer Aspire One

Video Unavailable

I have never heard of Kuki Linux before watching this video on how to install it on the Acer Aspire One netbook. Kuki Linux is a lightweight Linux distribution based on Ubuntu using the Xfce desktop environment. It is built especially for the Acer Aspire One.

Kuki Linux was founded by João Ferro from Portugal as a replacement for the Linpus Lite distro that is pre-installed on Linux models of the One. The installation of Kuki takes about 18 minutes. It boots pretty quickly and is claimed to run stable and fast.
