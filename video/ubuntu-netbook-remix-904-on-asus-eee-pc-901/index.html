Ubuntu Netbook Remix 9.04 on ASUS Eee PC 901

This video is a screen recording of Ubuntu Netbook Remix 9.04 (Jaunty Jackalope) running on an ASUS Eee 901. The video was recorded in full screen mode without sound and demonstrates applications like Nautilus file browser, Pidgin instant messenger, XChat IRC client, OpenOffice.org Writer, Firefox Web browser, the System Monitor, Rhythmbox music player as well as connecting to a mobile phone via bluetooth.
