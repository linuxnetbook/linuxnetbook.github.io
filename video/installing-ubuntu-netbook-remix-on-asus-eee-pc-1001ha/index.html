Installing Ubuntu Netbook Remix on ASUS Eee PC 1001HA

Here is another tutorial style video from the OSGUI Tech Show showing the installation of Ubuntu Netbook Remix 9.10 on an ASUS Eee PC 1001HA netbook from a USB drive.

To learn how to create a bootable USB drive for running or installing Ubuntu Netbook Remix check out the Ubuntu documentation page Installation From USB Stick.

One of the nice things about the bootable USB drive is, you can test whether your hardware is supported and see how you like the operating system without installing it to your computer.
