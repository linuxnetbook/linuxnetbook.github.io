Dell 3100 Touchscreen Chromebook VH5H8

Brand: Dell
Operating System: Chrome OS
Display: 11.6" HD (1366x768)
Dimensions (WxDxH): 1.0" x 1.0" x 1.0"
Touch Screen: Yes
Weight: 4.0 lbs
Processor: 1.1 GHz Intel Celeron N4000
Memory: 4 GB LPDDR4
Storage: 32 GB eMMC
WLAN: 802.11AC
Colors: Black
Input Devices: Keyboard, Touchpad, Webcam
Connectivity: Bluetooth, HDMI, USB, Wi-Fi

The Dell 3100 is a lightweight Chromebook tailored for everyday computing tasks. Released on November 29, 2019, this device is powered by an Intel Celeron N4000 processor, boasting a clock speed of 1.1GHz and 2 cores. Its compact design includes an 11.6" HD touchscreen display with a resolution of 1366 x 768, making it suitable for both work and personal use.

Key Features

  • Processor: Intel Celeron N4000
  • RAM: 4GB LPDDR4
  • Storage: 32GB eMMC
  • Display: 11.6" HD touchscreen
  • Connectivity Options:
  • Bluetooth 5.0
  • 2 USB Type-C ports
  • 2 USB 3.1 ports
  • HDMI
  • Wi-Fi (Intel Dual Band Wireless AC 9560)
  • Input Devices: Keyboard, Touchpad, Webcam

This Chromebook is an ideal option for users seeking a portable and functional device, offering reliable performance for web browsing, document editing, and media consumption. While it features a lightweight design at only 4 pounds and a depth of 1.0 inches, it manages to pack in essential features suited for diverse computing needs.

Whether you need a secondary device for travel or a primary machine for light computing tasks, the Dell 3100 provides an efficient solution with its balance of performance and portability.
