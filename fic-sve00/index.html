FIC SVE00

Brand: FIC
Operating System:
Display: 10.1" LED Widescreen (1024x600)
Dimensions (WxDxH): 11.2" x 8.5" x 1"
Weight: 2.65 lbs
Processor: 1.66 GHz Intel Atom N450
Memory: 1 GB DDR2 SDRAM (max: 2 GB)
Storage: 32 GB SSD
WLAN: WiFi 802.11 b/g/n
LAN: 10/100/1000 Ethernet
Colors: Ruby Red, Urban White
Input Devices: Keyboard, Touchpad, Microphone, Video Camera
Connectivity: USB, SD Card reader

The SVE00 is a 10.1" netbook by Taiwan based First International Computer, Inc. (FIC) powered by a 1.66GHz Intel Atom N450 processor. It comes with either SSD (up to 32GB) or HDD (up to 320GB), can be equipped with 2GB DDR2 RAM, has a 1.3MP webcam, 2 USB 2.0 ports, 4-in-1 card reader, microphone, security lock, an embedded 3 cell battery, and a waterproof keyboard.

The netbook weighs about 2.65lbs (1.2kg) and measures 11.2 x 8.5 x 1 inches (285 x 215 x 26.5mm). It comes with built-in WiFi 802.11b/g/n and 10/100/1000 Ethernet and offers optional support for Bluetooth, WiMAX, and HSDPA.

The SVE00 is compatible with Windows 7, XP, and Linux. So far,I there is no information on pricing, availability, and whether retailers will ship this netbook with a Linux distro pre-installed. I'll update this page in case the latter happens.

For further information see FIC's press release and product page with some high resolution images and product specifications that can be downloaded in PDF format.
