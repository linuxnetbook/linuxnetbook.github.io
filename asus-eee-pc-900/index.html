ASUS Eee PC 900

Brand: ASUS
Operating System: Xandros
Display: 8.9" TFT LCD (1024x600)
Dimensions (WxDxH): 8.9" x 6.5" x 0.9"
Weight: 2.2 lbs
Processor: 0.9 GHz Intel Celeron
Memory: 1 GB DDR2
Storage: 20 GB SSD
Battery life: 3.5 hours
WLAN: WLAN 802.11b/g
LAN: Ethernet 10/100/1000 Mbit/s
Colors: Shiny Black, Shiny White, Purple
Input Devices: Keyboard, Touchpad, Microphone, Video Camera
Connectivity: USB, SD Card reader, Expresscard 54 slot

The ASUS Eee PC 900 is equipped with an Intel Celeron M 900 MHz processor and offers a wide range of options for mobile computing. Eee's shockproof design, a weight of 2.2 pounds (less than 1kg), dimensions of 8.9 x 6.5 x 0.9 inches (22.5 x 17 x 2 cm), and the energy saving Solid State Disk (SSD) technology make this netbook an optimal outdoor companion anywhere, anytime.

The ASUS Eee PC 900 20G features a 8.9 inch widescreen, a 1.3M pixels webcam, and 20GB storage capacities for GNU Linux operating systems.

The Linux version runs Xandros, which is a Debian-based distribution created by the Xandros Corporation, which also offers professional support. Xandros features popular applications like OpenOffice.org, Firefox, Thunderbird, and is well-suited to be run in a mixed environment with Windows. The Xandros GUI is a modified version of KDE.
