Always Innovating Touch Book

Brand: Always Innovating
Operating System:
Display: 8.9" Tochscreen (1024x600)
Dimensions (WxDxH): 9.4" x 7" x 1.4"
Weight: 2 lbs
Processor: 0.8 GHz TI OMAP3530
Memory: 256 MB Flash Memory
Battery life: 10 hours
WLAN: Wifi 802.11 b/g/n
Colors: Black
Input Devices: Keyboard, Touchpad, Microphone
Connectivity: Bluetooth, USB, SD Card reader

The Touch Book by Always Innovating combines netbook and touchscreen tablet in one device. It has an 8.9" pressure sensitive touchscreen with a maximum resolution of 1024x600 pixels and is powered by a 600MHz Texas Instruments OMAP3530 ARM processor with Micron 256MB (RAM) + 256MB (NAND) Memory.

The Touch Book comes with an 8GB SD card as main storage, internal USB WiFi 802.11 b/g/n , internal USB bluetooth class 2.1, a US Qwerty 24cm-large keyboard (around 95% standard keyboard size), two Owolff high-quality internal stereo speakers, 3D accelerometer, and two internal 6000 and 12000 mAh batteries for about 10 hours battery life.

The Touch Book runs a custom Linux operating system built on Linux kernel version 2.6.29 and the OpenEmbedded framework. It uses the Xfce 4.4 desktop environment, features a unique 3D interface for starting applications with your finger, and applications like Firefox 3.0, Abiword and Gnumeric, Mypaint 5.1, evince PDF reader, Pidgin instant messenger. The software offers support for most standard USB accessories and Java based applications. Moreover Always Innovating plans to support other Linux based alternatives such as Google Android or Moblin.

The Touch Book tablet costs US $299 and the tablet with keyboard $399.00. Both are currently available for pre-order. Initial shipment is expected for July 2009, so not much time left to meet the targeted deadline. The software is announced as work-in-progress during summer 2009 with frequent automatic updates and support to be available.
